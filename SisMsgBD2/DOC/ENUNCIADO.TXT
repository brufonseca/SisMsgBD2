   SISTEMA MENSAGENS (versao usando BD) (SISmsgBD) com Delphi acessando o BD  e mostrando na tela (IU)

Enunciado:  Desenvolver um sistema SISmsgsBD para atender o seguinte minimundo:
 Temos um BD  com tabela de mensagens textuais para serem exibidas convertidas segundo um "codigo secreto" e gravadas como mensagem convertida na mesma tabela de mensagens.
 Fazer um módulo CONVERTE de conversão que use um código de conversão reversível, ou seja,
  ( Converte(Converte(msg))=msg ) . USAR UMA IMAGEM PARA ILUSTRAR A MSG

O Projeto deve estar em uma pasta SISmsgsBD com os subdiretorios 
  DADOS, DOC e MODULOS

I) Projeto Conceitual do BD  
Resposta: Apenas uma Classe de mensagens com atributos numero_da_mensagem, texto, autor e mensagem_convertida. Na interface, usar um formulario onde serão apresentadas sequencialmente as mensagens existentes para escolha (O sistema deve permitir tambem inclusões de novas mensagens). A mensagem escolhida poderá ser convertida, mostrada e sua versão convertida gravada no BD

II) a)Projeto Lógico do BD
 Um BD sqlite MSGS.DB com uma tabela MSGSS(NUM integer, MSG text, AUTOR text, MSGC text); 
 onde o texto MSGC será o resultado de CONVERTE(MSG).
   
   b)Projeto Lógico da IU
    b1)UM form principal que contem dois memos (memo1 e memo2) para as mensagens de entrada
    e saida. O form tambem tem botões para controlar o processamento das mensagens

   c)Projeto Lógico do Processo de conversão
     Módulo de Conversão da mensagem do 1o memo para ser gravada no 2o memo
     A conversão será como segue:

      Usar Cod=´gaderipolutyGADERIPOLUTY' que é um Código reversível. (Poderia ser outro...)
      O modulo vai codificando cada caracter do texto do memo1 segundo a lógica abaixo:

      PARA CADA char do texto do memo1	FACA 
       SE char esta no Codigo Cod 
	ENTAO 
         SE posicao do char é par ENTAO troca-o pelo anterior 
		                  SENAO troca-o pelo posterior 
         FIM_SE	
        SENAO (nao troca)
       FIM_SE
       grava-o no texto do Memo2 de saida. 
      FIM-PARA
